## âœ¨ Functionality

The project includes three main tasks demonstrating different JavaScript techniques:

### Task 1: URL Slug Generator
- **Function**: `slugify(title)`
- **Description**: Converts article titles into URL-friendly slugs
- **Features**:
  - Converts text to lowercase
  - Replaces spaces with hyphens
  - Creates SEO-friendly URLs

### Task 2: Array Merger
- **Function**: `makeArray(firstArray, secondArray, maxLength)`
- **Description**: Combines two arrays with a maximum length constraint
- **Features**:
  - Concatenates arrays
  - Limits result to specified maximum length
  - Returns sliced array when necessary

### Task 3: Number Filter
- **Function**: `filterArray(numbers, value)`
- **Description**: Filters array elements greater than a threshold value
- **Features**:
  - Iterates through numeric arrays
  - Filters values based on comparison
  - Returns new filtered array
